Git Commands Vocabulary

This covers the top 10 git commands that are required in day to day life.

Git Commandth> Discription
Git add Adds changes for the whole directory to staging but without deleting files
Git pull Obtains the most recent changes from the origin and merges them
Git status Displays the status of modified files
Git fetch Displays the status of modified files
Git remote Displays the name of remote repositories
Git clone Creates a copy of an existing Git repository
Git commit Takes the staged snapshot and commits it to the project history
Git config A convenient way to set configuration options for your Git installation
Git init Initializes a new Git repository
Git log Lets you explore the previous revisions of a project
